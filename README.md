# Meu Portfólio

Este é o repositório do meu portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e formas de contato.

## Demo
Acesse aqui: [https://matheus-dev-portfolio-two.vercel.app/](https://matheus-dev-portfolio-two.vercel.app/)

## Sobre o projeto
O portfólio foi desenvolvido com foco em apresentar minha trajetória na área de tecnologia, meus principais projetos e minhas habilidades como desenvolvedor.

## Tecnologias utilizadas
- Next.js
- React
- TypeScript
- NodeJs
- CSS
- Vercel

## Como executar o projeto localmente

```bash
